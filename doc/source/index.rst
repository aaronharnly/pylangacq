PyLangAcq: Language Acquisition Research in Python
==================================================

PyLangAcq is a Python library primarily for language acquisition research.

Features:

* Comprehensive capabilities of handling CHAT transcripts
  as used in CHILDES
* Standard analytic tools such MLU and IPSyn readily available
* Benefits from Python: fast coding, numerous libraries for computational
  modeling and machine learning
* Use of CHAT: Powerful research with conversational data in general

PyLangAcq is under active development. The current version is 0.6.
